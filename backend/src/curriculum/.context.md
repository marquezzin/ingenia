# App: curriculum

## Propósito
Estrutura de conteúdo pedagógico da plataforma: módulos, aulas, videoaulas, exercícios e casos de teste para correção automática.

## Models
- **`Module`** — Unidade temática da trilha. Campos: `title`, `description`, `sequence_order` (unique), `publication_status`
- **`Lesson`** — Aula dentro de um módulo. Campos: `module` (FK), `title`, `written_content`, `sequence_order`, `publication_status`. Unique: `(module, sequence_order)`
- **`VideoLesson`** — Videoaula 1:1 com Lesson. Campos: `lesson` (O2O), `title`, `video_url`, `duration_seconds` (null)
- **`Exercise`** — Exercício prático vinculado a uma aula. Campos: `lesson` (FK), `title`, `statement`, `support_message` (null), `sequence_order`, `publication_status`. Unique: `(lesson, sequence_order)`
- **`ExerciseTestCase`** — Caso de teste para correção automática. Campos: `exercise` (FK), `name`, `input_data` (null), `expected_output`, `sequence_order`, `is_hidden`. Unique: `(exercise, sequence_order)`

## Enums (`enums.py`)
- **`ContentStatus`**: DRAFT, PUBLISHED, ARCHIVED

## Regras de Negócio
- **BR-006**: Ordenação única de módulos por `sequence_order` (unique constraint)
- **BR-007**: Ordem das aulas é única dentro de cada módulo (unique_together)
- **BR-008**: Estrutura pronta para validar vídeo + conteúdo ao publicar (VideoLesson 1:1)
- **BR-009**: Exercício vinculado a uma aula via FK (not nullable)
- **BR-010**: Estrutura pronta para validar ao menos um test case ao publicar

## Endpoints

### Module CRUD (admin only) — `api/v1/modules/`
| Método | URL | Descrição |
|--------|-----|-----------|
| GET | `/api/v1/modules/` | Lista paginada, filtro por `publication_status`, busca por `title`, ordenação por `sequence_order` |
| POST | `/api/v1/modules/` | Criar módulo |
| GET | `/api/v1/modules/:id/` | Detalhe (inclui `lesson_count`) |
| PUT | `/api/v1/modules/:id/` | Atualizar módulo |
| DELETE | `/api/v1/modules/:id/` | Remover módulo |

### Lesson CRUD (admin only) — `api/v1/modules/:moduleId/lessons/`
| Método | URL | Descrição |
|--------|-----|-----------|
| GET | `/api/v1/modules/:moduleId/lessons/` | Lista paginada, filtro por `publication_status` |
| POST | `/api/v1/modules/:moduleId/lessons/` | Criar aula (com VideoLesson inline opcional) |
| GET | `/api/v1/modules/:moduleId/lessons/:id/` | Detalhe (inclui `video` nested e `exercise_count`) |
| PUT | `/api/v1/modules/:moduleId/lessons/:id/` | Atualizar aula + VideoLesson inline |
| DELETE | `/api/v1/modules/:moduleId/lessons/:id/` | Remover aula (cascade deleta vídeo) |

## Services
- **`CreateModuleUseCase`** — Cria módulo com validação de `sequence_order` único
- **`UpdateModuleUseCase`** — Atualiza módulo com validação de `sequence_order` único (excluindo o próprio)
- **`DeleteModuleUseCase`** — Remove módulo por ID
- **`CreateLessonUseCase`** — Cria aula com validação de módulo existente e `sequence_order` único no módulo. Cria VideoLesson inline se fornecido
- **`UpdateLessonUseCase`** — Atualiza aula + gerencia VideoLesson (criar/atualizar/remover conforme payload)
- **`DeleteLessonUseCase`** — Remove aula por ID (cascade deleta vídeo)

## Selectors
- `get_module_by_id(*, id)` — Retorna Module por ID
- `list_modules()` — Lista todos os módulos ordenados por `sequence_order`
- `get_lesson_by_id(*, id)` — Retorna Lesson por ID
- `list_lessons_for_module(*, module_id)` — Lista aulas de um módulo ordenadas por `sequence_order`

## Serializers
- **`ModuleListSerializer`** — Campos resumidos: id, title, sequence_order, publication_status
- **`ModuleDetailSerializer`** — Todos os campos + lesson_count
- **`ModuleCreateUpdateSerializer`** — Campos de escrita para criação/edição
- **`VideoLessonSerializer`** — Leitura: id, title, video_url, duration_seconds
- **`VideoLessonInputSerializer`** — Escrita inline: title, video_url, duration_seconds (optional)
- **`LessonListSerializer`** — Campos resumidos: id, title, sequence_order, publication_status, has_video
- **`LessonDetailSerializer`** — Todos os campos + video nested + exercise_count
- **`LessonCreateUpdateSerializer`** — Campos de escrita + video_lesson nested (optional)
