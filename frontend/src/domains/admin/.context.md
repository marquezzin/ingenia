# Admin Domain — .context.md

## Propósito

Domain frontend para o painel administrativo da plataforma. Permite ao administrador visualizar métricas, gerenciar módulos, aulas, exercícios e usuários.

## Páginas

| Página | Rota | Descrição |
|--------|------|-----------|
| `DashboardPage` | `/admin` | Visão geral com stat cards e atalhos rápidos |
| Módulos (futuro) | `/admin/modules/*` | CRUD de módulos (ISSUE-010-B) |
| Usuários (futuro) | `/admin/users/*` | CRUD de usuários (ISSUE-010-D) |
| Turmas (futuro) | `/admin/classes` | Visão read-only de turmas (ISSUE-010-E) |

## API Contract

| Função | Endpoint | Descrição |
|--------|----------|-----------|
| `getAdminDashboardStatsApi` | `GET /api/v1/admin/stats/` | Estatísticas agregadas do dashboard |

## Hooks

| Hook | Descrição |
|------|-----------|
| `useAdminDashboardStats` | `useQuery` para carregar stats do dashboard |

## Types

| Tipo | Descrição |
|------|-----------|
| `AdminDashboardStats` | Resposta do endpoint de stats (totais de módulos, aulas, exercícios, usuários) |

## Dependências

- `@/shared/ui/components` — `StatCard`, `PageHeader`
- `@/shared/http/client` — `httpClient`
- `@tanstack/react-query` — hooks de dados

## Backend Correspondente

- `curriculum` app — endpoint `/api/v1/admin/stats/` (view `AdminDashboardStatsView`)
- `accounts` app — endpoint `/api/auth/admin/users/` (gestão de usuários)
